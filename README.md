# EchoParrot-A (Beta) — Tab Audio Recorder v1.0

現在アクティブなChromeタブの音声をワンクリックで録音・保存できるChrome拡張機能です。

## 機能

- 現在のアクティブタブの音声のみを録音
- リアルタイム波形表示（録音中に視覚的フィードバック）
- 録音時間タイマー表示
- WebM形式（audio/webm;codecs=opus）で自動保存
- **Google Gemini API 対応フォーマット**



## インストール方法

1. このリポジトリをダウンロード（ZIP展開 or clone）
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このフォルダを選択

## 使い方

1. 録音したいタブを開く
2. 拡張機能アイコンをクリックしてポップアップを開く
3. **「● 録音開始」** をクリック
4. 録音中は波形と赤い点滅インジケーターが表示される
5. **「■ 停止・保存」** をクリックするとWebMファイルが自動ダウンロード

## 注意事項

- タブの音声を録音するには、対象タブで音声が再生されている必要があります
- `tabCapture` 権限のため、拡張機能ポップアップからのみ録音開始が可能です
- 保存ファイル名: `tab-recording-YYYYMMDD-HHmmss.webm`
